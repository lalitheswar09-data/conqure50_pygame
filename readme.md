**Conquer 50** is a turn-based **strategy board game** built using **Python and Pygame**, where players compete to strategically claim territory and reach the target score of 50.  
The game focuses on logical decision-making, turn management, and grid-based gameplay.

---

##  Features
- Turn-based gameplay  
- Strategic point accumulation  
- Interactive graphical interface using Pygame  
- Mouse-based input handling  
- Real-time game state updates  
- Fully offline gameplay (no APIs used)

---

##  Game Logic Overview
The game revolves around:
- Strategic move selection  
- Score tracking and updates  
- Turn switching between players  
- Win-condition detection based on reaching the target score  

Each move impacts the board state and overall score, encouraging careful planning.

- **Language:** Python  
- **Graphics Library:** Pygame
